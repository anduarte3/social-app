# social-app

This is a full-stack web application that enables users to connect in a social networkâ€“style environment. Users can create accounts, share posts, like and comment on others posts, and manage their own content. The app is built with a responsive design and secure authentication.

---
## Features
- User registration and secure authentication (JWT with Passport.js)
- Login/logout functionality
- Create and delete posts
- Like and comment on posts
- Fully responsive UI with Tailwind CSS

## Tech Stack & Tools
- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: Passport.js with JWT
- **Realtime**: Socket.io
